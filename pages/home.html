Layanan Supernova Studio



Rental Studio Musik

Rental Studio Musik adalah Penyewaan Set Alat musik di dalam ruangan, sangat cocok untuk kalian yang ingin melakukan latihan karena harga sewa setiap jamnya sangat terjangkau untuk pelajar.

Rental Alat Musik

Rental Alat Musik adalah Penyewaan Set Alat musik di luar ruangan, biasanya untuk acara contohnya adalah acara pernikahan, acara pentas seni disuatu sekolah.

Studio Rekaman

Studio rekaman adalah fasilitas khusus untuk perekaman suara, pencampuran, dan produksi audio pertunjukan musik instrumental atau vokal, kata-kata yang diucapkan, dan suara lainnya.
